## Dependencies

### Arch
```
sudo pacman -S wine jq curl
```

### Debian
```
sudo apt install wine jq curl
```

## Install Magic

❗❗ Do not change the folder-paths on Windows ❗❗

```
git clone https://github.com/linux-ott/mtga-launcher.git
```
```
cd mtga-launcher/
```
```
./mtga-launcher
```

## Force Magic
```
mtga-launcher --update 
```
